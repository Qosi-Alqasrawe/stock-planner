Stock Planner

A simple planning dashboard that converts Excel stock data into clear production and management decisions.

What This Tool Does

Combines Stock and Items Excel files into one clean planning view

Calculates demand, coverage days, and risks

Flags critical products and customer shortages

Builds a machine-based production plan with suggested quantities

Allows planners to confirm Final Qty

Automatically fills the official Production Plan.xlsx

Generates management reports (Excel + PDF)

Key Outputs

Master Planning Table

Product Alert & Customer Alert

Production Plan per Machine

Final Production Plan (ready for execution)

Management Summary Report

How to Run
pip install -r requirements.txt
streamlit run app.py

How to Use

Upload Stock.xlsx

Upload Items.xlsx (optional)

Review alerts and production plan

Confirm Final Qty

Auto-fill Production Plan.xlsx

Download reports for management
